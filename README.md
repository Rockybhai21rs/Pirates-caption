# Auto Caption Bot

A Telegram bot that automatically edits video captions when posted in a channel.

## Deployment on Koyeb

1. Fork this repository.
2. Create a new Koyeb service and connect this repo.
3. Set environment variables:
   - `API_ID`
   - `API_HASH`
   - `BOT_TOKEN`
